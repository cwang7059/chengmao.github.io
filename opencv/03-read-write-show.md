# 03 - 读写与显示（把练习跑起来）

## 读图 / 写图

- 读图：`imread`
- 写图：`imwrite`

建议新手练习都“落盘”保存结果图，避免环境没有 GUI 导致 `imshow` 用不了。

## 显示（可选）

- `imshow` + `waitKey`：用于桌面环境快速看结果
- WSL/无桌面：优先用 `imwrite` 输出到文件

## 色彩空间（最常用）

- BGR ↔ Gray：做阈值/边缘/轮廓常用
- BGR ↔ HSV：做颜色分割常用

**注意**：OpenCV 默认 BGR，不是 RGB。

## 最小练习清单

1. 读一张图，分别保存：原图、灰度图
2. 把图缩放到固定宽度（比如 640），保存
3. 转 HSV 后，提取一个颜色范围做 mask（比如红色/蓝色），保存 mask
