# OpenCV 新手快速上手（目录）

如果你是第一次系统学 OpenCV，建议严格按顺序刷完这套文档：先把“图像/矩阵的基本模型”打牢，再进入常用算子与小项目。

## 学习路线（建议顺序）

1. `opencv/00-getting-started.md`：你要学什么、怎么练、怎么避坑
2. `opencv/01-install.md`：安装与环境（Python / C++）
3. `opencv/02-mat-and-image-basics.md`：图像=矩阵、通道、类型、ROI
4. `opencv/03-read-write-show.md`：读写显示、色彩空间、基本像素操作
5. `opencv/04-filtering.md`：滤波与去噪（均值/高斯/中值/双边）
6. `opencv/05-edges-threshold-morphology.md`：边缘/阈值/形态学（做二值处理的核心）
7. `opencv/06-contours-and-geometry.md`：轮廓、外接矩形/圆、面积周长、拟合
8. `opencv/07-features-and-matching.md`：特征点与匹配（入门到够用）
9. `opencv/08-video-and-camera.md`：视频流/相机、常见坑
10. `opencv/09-mini-projects.md`：小项目清单（练到能做“完整 pipeline”）
